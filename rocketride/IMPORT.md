# Import `finterms.pipe`

1. Open [`../finterms.pipe`](../finterms.pipe) in Cursor (Rocket Ride canvas)
2. Or use Pipeline Builder **Import** and select that file
3. Open any **LLM** node → confirm API key / `${ROCKETRIDE_OPENAI_KEY}`
4. **Play** on **Chat** to test the multi-stage + Agent path
5. Website uses the **Webhook** path automatically via the SDK (no Play needed)
