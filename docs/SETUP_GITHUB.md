# GitHub setup (beginner)

Yes — when you create the repo, **paste the URL into this chat** (example: `https://github.com/yourname/FinTerms`). I will connect this project and push the code for you.

## Step-by-step

### 1. Create a GitHub account (if you don’t have one)
1. Go to [https://github.com/signup](https://github.com/signup)
2. Create an account and verify your email

### 2. Create a new empty repository
1. Go to [https://github.com/new](https://github.com/new)
2. **Repository name:** `FinTerms` (or anything you like)
3. Set to **Public** (easier for hackathon judges) or Private if required
4. **Do not** check “Add a README”
5. **Do not** add .gitignore or license (we already have files locally)
6. Click **Create repository**

### 3. Copy the URL
On the new repo page, copy the HTTPS URL, like:
`https://github.com/YOUR_USERNAME/FinTerms.git`

### 4. Paste it here
Paste that URL in chat and say “please push”.

I will:
- initialize git if needed
- commit the project (**without** your `.env` / API key)
- push to GitHub

### 5. (Later) You may be asked to sign in
If push asks for login:
- Prefer **GitHub CLI** (`gh auth login`) or a **Personal Access Token** when Git asks for a password
- Never put tokens into chat

---

**You do not need to learn git commands for this hackathon** unless you want to.
